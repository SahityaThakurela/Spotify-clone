### Spotify Clone

Spotify Clone is a web-based application that replicates the core user interface and music playback functionality of the popular music streaming platform, Spotify. 

The project is built entirely with vanilla web technologies, demonstrating a strong understanding of DOM manipulation, HTML5 Audio integration, and complex CSS layouts (Flexbox/Grid) to create a seamless and interactive user experience.

Key Features:

**UI/UX Replication**: Closely mimics the design, color scheme, and layout of Spotify, including the sidebar, main content area, and bottom playback bar.

**Audio Playback**: Fully functional playback controls allowing users to play, pause, skip forward/backward, and adjust volume using the HTML5 Audio API.

**Dynamic Rendering**: Utilizes JavaScript to dynamically load song libraries, update the currently playing track details (cover art, song name, artist), and manage a functional progress bar.

**Responsive Design**: Styled to provide a smooth and visually appealing experience that adapts to different screen sizes, ensuring accessibility on both desktop and mobile devices.

Tech Stack:

**Frontend**: HTML5, CSS3, JavaScript (Vanilla ES6+)

**Media**: HTML5 Audio API

**Deployment**: Hosted live on Vercel
